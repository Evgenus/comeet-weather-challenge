The best weather on earth

Web development - Exercise brief

Where's the best weather on earth right now?

Use the OpenWeather JSON API and create a simple page to show which weather station (or city) has the most convenient weather in terms of temperature and humidity (the ideal is 21Â°C, 50%). Below the winning weather station, show the following leading ones.

Extra: Allow to choose whether to display the optimal place for a male or a female (one Celsius degree more).

Requirements

  • The page will work on client-side only, will be tested only in Google Chrome
  • Use AngularJS as a framework. You may use any other framework or library you are comfortable with. We recommend: moment.js (for time manipulations), underscore.js (for sorting), Less (for css)
  • Emphasize simple, structured and reusable code
  • An elegant usable page with clear information hierarchy
  • Don't worry about performance, but preferably let the user know what's happening

Hint: Use the bounding box version of the API
Time: 4 hours, no more
